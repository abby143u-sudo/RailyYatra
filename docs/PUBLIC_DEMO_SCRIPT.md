# RailYatra Public Demo Script

## Positioning

Say this clearly:

RailYatra is a real railway route recommendation preview.

Do not say:

- live booking app
- official railway ticketing app
- payment-enabled ticketing app
- PNR app
- live seat availability app
- live fare app

## Demo flow

1. Open the frontend.
2. Show the public beta warning banner.
3. Explain that live booking, PNR, payment, cancellation, live fare and live availability are not connected yet.
4. Show the product status panel.
5. Show the public beta checklist panel.
6. Search a sample route in the search-v2 preview.
7. Search a sample route in the recommend-v2 preview.
8. Explain rank, confidence, transfer safety and reasons.
9. Show station suggestions.
10. Show train stop drilldown.
11. Close with next roadmap: live integrations, official APIs, production auth and deployment hardening.

## Sample route for demo

Try:

- Source: LTT
- Destination: VVH

If this route returns no result, use station suggestions and choose another pair from real staging data.

## One-line pitch

RailYatra helps users discover smarter railway route options using real railway route data, ranked recommendations, confidence labels and transfer-safety explanation.

## What is ready

- real staging data
- search-v2
- recommend-v2
- station suggestions
- train stop drilldown
- product status
- beta checklist
- deployment configuration

## What is not ready

- live booking
- live fare
- live availability
- PNR
- payment
- cancellation
- official ticketing claim

## Final safety line

This is a railway route recommendation preview, not a live ticket booking service yet.
