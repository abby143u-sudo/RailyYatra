# RailYatra Final Demo Script

## 30-second intro

RailYatra is a railway route recommendation preview. It helps users discover possible rail journey options, compare direct and safer alternatives, and understand route tradeoffs before booking elsewhere.

## Demo flow

1. Open the live frontend.
2. Show the public boundary: booking, payment, PNR and live availability are not connected yet.
3. Search PNBE to NDLS.
4. Show the route result.
5. Explain that RailYatra can rank routes by directness, transfer safety and route confidence.
6. Show feedback and analytics persistence.
7. Open admin preview with protected token.
8. Show admin audit logs.

## What is real today

- Live deployed frontend.
- Live deployed backend.
- Route recommendation API.
- Feedback persistence.
- Analytics persistence.
- Protected admin auth.
- Admin audit logs.

## What is not connected yet

- IRCTC booking.
- Payments.
- PNR.
- Real seat availability.
- Real fare.
- Real live train running status provider.
