# RailYatra Public Demo Test Script

Use this script while showing the public demo.

## 1. Open demo

https://raily-yatra.vercel.app

## 2. Say this positioning

RailYatra is a real railway route recommendation preview. It uses staging railway data to show route options, ranked recommendations and transfer-safety signals.

## 3. Safety line

This is not a live ticket booking service yet. Live booking, payment, PNR, live fare and live seat availability are not connected.

- Direct route recommendations

- Transfer safety
- Why recommended
- Booking status warning

## 6. Closing line

RailYatra is currently at public demo preview stage. The next step is production-grade live integrations and user-facing polish.
